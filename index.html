FlexDM

Flexible Data Mining using WEKA

Download this project as a .zip file Download this project as a tar.gz file

Performing massive data mining experiments with multiple datasets and methods is a common task faced by many research groups. WEKA is a machine learning package designed to facilitate this task by providing tools that allow researchers to select from a plethora of classification methods and specific test strategies. Despite its popularity, the current WEKA Experimenter, has limitations that impact its usability: lack of flexibility for parameter ranges; difficult setup for multi-core parallelisation; results are not saved until the last test has concluded by default; and the XML schema is difficult to read. FlexDM implements an interface with WEKA to run batch processing tasks in a simple and intuitive way. In an XML file hundreds of tests can be defined to run on several datasets. FlexDM allows those tests to be executed in parallel to take advantage of multi-core processors, significantly increasing usability. Results are saved incrementally for better robustness and reliability.
